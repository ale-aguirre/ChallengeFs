## Configuración de desarrollo

Crear un archivo .env en api , para declarar:
DB_USER= 'usuario en postgres'
DB_PASSWORD= 'contraseña en postgres'
DB_HOST=localhost

```sh
cd api
npm install
npm start
```

```sh
cd client
npm install
npm start
```
